.X4

A very fast ZSH Prompt.

.X4 consists of modular components and dotfiles that serve to make daily life in a shell easier. Trying to squeeze more juice, by every commit.

Just clone this repo, then run ./install, no worries it asks before doing anything.

git clone --recursive https://github.com/X4/.X4.git "${ZDOTDIR:-$HOME}/"

Note: To get familiar with aliases just type aliases=<TAB> and ZSH will list all available aliases for you. All aliases are namespaced, that means for you that ie. g.<TAB> will complete all git aliases.
